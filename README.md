# 🌍 Travel Spot Management System

**OS Course Project** — Interactive menu-driven Bash script replicating a travel website.

**Student:** SHIMANTO SHIRSHO (2204009)  
**Course:** Operating Systems, CSE Department  

## How to Run
```bash
chmod +x travel_manager.sh
./travel_manager.sh
```

## Requirements
- Linux / macOS terminal (Bash)
- On Windows: use **WSL** (Windows Subsystem for Linux)

## Features
1. Add / list travel spots
2. Search by city or country
3. Add / view ratings
4. Add / view hotels
5. Full spot summary with average ratings
6. Export report to `summary_report.txt`

## Files
| File | Purpose |
|------|---------|
| `travel_manager.sh` | Main script |
| `spots.txt` | Travel spots database |
| `ratings.txt` | User ratings database |
| `hotels.txt` | Hotels database |
